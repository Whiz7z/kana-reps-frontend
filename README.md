# KanaReps Web

**Documentation only** — no application code in this folder. Use it when you scaffold the real Vite app elsewhere.

Target stack: React + TypeScript + Vite + Tailwind. See the [documentation index](../docs/OVERVIEW.md) and [docs/BACKEND_INTEGRATION.md](docs/BACKEND_INTEGRATION.md).

This folder is intended to become its **own Git repository** when you implement.

## Environment (when you implement)

Point `VITE_API_URL` at your running API (see [kana-reps-api](../kana-reps-api)).

## Environment

| Variable | Description |
|----------|-------------|
| `VITE_API_URL` | Base URL of the API (no trailing slash) |
