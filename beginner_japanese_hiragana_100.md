# 100 Beginner Japanese Words (Hiragana)

This list contains 100 essential Japanese words for beginners, focusing on those commonly written in Hiragana or useful for early learners.

| Hiragana | Romaji | Meaning |
| :--- | :--- | :--- |
| おはよう | Ohayou | Good morning (informal) |
| こんにちは | Konnichiwa | Hello / Good afternoon |
| こんばんは | Konbanwa | Good evening |
| さようなら | Sayounara | Goodbye |
| ありがとう | Arigatou | Thank you |
| はい | Hai | Yes |
| いいえ | Iie | No |
| すみません | Sumimasen | Excuse me / I'm sorry |
| ごめんなさい | Gomen nasai | I am sorry |
| おねがいします | Onegaishimasu | Please (polite) |
| どうぞ | Douzo | Here you go / After you |
| おやすみ | Oyasumi | Goodnight (informal) |
